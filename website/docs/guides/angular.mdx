# Angular

---

To include Fontsource into your Angular project, you can import the Fontsource files into your project's css files.

<CarbonAd />

## Installation

---

_app.component.ts_

```ts
import { Component } from "@angular/core";

@Component({
  selector: "app-root",
  templateUrl: "./app.component.html",
  styleUrls: ["./app.component.css"], // file with Fontsource imports
})
export class AppComponent {
  title = "my-app";
}
```

_app.component.css_

```css
@import "~@fontsource/open-sans/400.css";
@import "~@fontsource/open-sans/700.css";
```
